## Instructions pour bien démarrer ( ^_^)／

Tout d'abord installez le bordel:

```
npm install
```

Puis faites un:

```
npm run gameinstall
```


## Instructions pour lancer le bordel 

Le front se lance sur http://localhost:8080... si tout va bien •ᴗ•.

Voila. 
That's all.
